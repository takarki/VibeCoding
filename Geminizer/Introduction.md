# Geminizer - Introduction

## 概要
Geminizerは、YouTube動画やWebページをGoogle Geminiで簡単に要約・分析できるChrome拡張機能です。ワンクリックまたはショートカットキーで、動画やページの内容をAIが解析し、理解しやすい形で提供します。

## 主な機能

### YouTube動画の要約
- **ワンクリック要約**: 動画ページに表示される「要約」ボタンをクリック
- **ショートカットキー**: `Ctrl+Shift+U` (Windows/Linux) / `Command+Shift+U` (Mac)で素早く実行
- **タイムスタンプ付き要約**: 詳細な要約と動画の各セクションへのリンク
- **自動字幕抽出**: 動画の字幕データを自動取得

### Webページの要約
- **3つの抽出モード**:
  - **URL**: リンクとタイトルのみを送信
  - **Text**: ページ本文を抽出して送信
  - **Capture**: スクリーンショットを取得
- **柔軟な分析**: ページの種類に応じて最適な方法を選択

### カスタムプロンプト
- **11種類のデフォルトプロンプト**:
  1. YouTube要約 - 詳細要約とタイムスタンプリンク
  2. インフォグラフィック構成案 - Canvas で視覚的な構成案
  3. NanoBanana Pro 図解作成 - 画像生成ツールで図解
  4. ポッドキャスト台本 - 会話形式の台本生成
  5. イメージ動画作成 - 動画制作の構成案
  6. 学習ガイド作成 - 段階的な学習ガイド
  7. 動画制作プラン - Veo 3.1 を使用した動画制作
  8. ガイド付き学習 - インストラクショナルデザインに基づくカリキュラム
  9. 📚 文系新卒1年目向け詳細解説 - 専門用語を初心者向けに解説
  10. 🔍 DeepResearch業界分析 - Deep Research ツールで業界分析
  11. 📝 詳細記事録作成 - 記事を構造化してナレッジベース化

- **プロンプトのカスタマイズ**: 自分専用のプロンプトを作成・保存可能
- **ショートカット設定**: 各プロンプトにカスタムショートカットキーを割り当て
- **競合検出**: ショートカットキーの重複を自動検出

## 技術スタック
- **フロントエンド**: Vanilla JavaScript
- **AI統合**: Google Gemini API
- **ストレージ**: Chrome Storage API
- **UI/UX**: カスタムCSS、直感的なインターフェース
- **YouTube API**: 字幕データの取得

## 対象ユーザー
- YouTube動画から効率的に情報を抽出したい学習者
- Webページの内容を素早く把握したいビジネスパーソン
- コンテンツの要約や再構成を行うクリエイター
- 研究や情報収集を効率化したい研究者
- 言語学習やスキルアップを目指す学生

## ユースケース
- **学習効率化**: 長い講義動画を短時間で理解
- **コンテンツ制作**: 動画内容から記事や台本を生成
- **リサーチ**: 複数の情報源を素早く調査・まとめ
- **知識管理**: 重要な情報をナレッジベースとして構造化
- **業界分析**: DeepResearch機能で包括的な市場調査
