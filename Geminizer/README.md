# Geminizer

YouTube動画やWebページをGeminiで簡単に要約・分析できるChrome拡張機能

![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

## 🌟 主な機能

### ✨ YouTube動画の要約
- 動画ページに表示される「要約」ボタンをワンクリック
- ショートカットキー（`Ctrl+Shift+U` / `Command+Shift+U`）で素早く実行
- タイムスタンプ付きの詳細な要約を自動生成

### 📝 Webページの要約
- URLとタイトルのみ、またはページ全体のコンテンツを抽出
- 3つのモード:
  - **URL**: リンクとタイトルのみを送信
  - **Text**: ページ本文を抽出して送信
  - **Capture**: スクリーンショットを取得

### 🎯 カスタムプロンプト
- 自分専用のプロンプトを作成・保存
- デフォルトで11種類の便利なプロンプトを用意:
  - **YouTube要約** - 動画の詳細要約とタイムスタンプリンク
  - **インフォグラフィック構成案** - Canvas で視覚的な構成案を作成
  - **NanoBanana Pro 図解作成** - 画像生成ツールで図解を作成
  - **ポッドキャスト台本** - 会話形式の台本を生成
  - **イメージ動画作成** - 動画制作の詳細な構成案
  - **学習ガイド作成** - 段階的な学習ガイド
  - **動画制作プラン** - Veo 3.1 を使用した動画制作の構成案
  - **ガイド付き学習** - インストラクショナルデザインに基づく学習カリキュラム
  - **📚 文系新卒1年目向け詳細解説** (NEW) - 専門用語を初心者にもわかりやすく解説
  - **🔍 DeepResearch業界分析** (NEW) - Deep Research ツールで包括的な業界・市場分析
  - **📝 詳細記事録作成** (NEW) - 記事を構造化してナレッジベース化

### ⌨️ ショートカットキー
- 各プロンプトにカスタムショートカットを設定可能
- 競合検出機能で安心
- デフォルト: `Ctrl+Shift+U` (YouTube要約)

### 🚀 高速・シンプル
- Geminiに直接プロンプトを送信
- バックグラウンドで動作し、元のタブに戻る
- 直感的なUI

## 📦 インストール

### Chrome Web Storeから（公開後）
1. [Chrome Web Store](https://chrome.google.com/webstore/)でGeminizerを検索
2. 「Chromeに追加」をクリック
3. 権限を確認して「拡張機能を追加」をクリック

### 開発版（手動インストール）
1. このリポジトリをクローンまたはダウンロード
```bash
git clone https://github.com/[username]/geminizer.git
```

2. Chromeで`chrome://extensions/`を開く

3. 右上の「デベロッパーモード」を有効化

4. 「パッケージ化されていない拡張機能を読み込む」をクリック

5. ダウンロードした`Geminizer`フォルダを選択

## 🎮 使い方

### YouTube動画を要約
1. YouTube動画ページを開く
2. 動画プレイヤー右上の「要約」ボタンをクリック
   - または `Ctrl+Shift+U` (Mac: `Command+Shift+U`) を押す
3. Geminiで要約が自動的に開始されます

### Webページを要約
1. 要約したいWebページを開く
2. 拡張機能アイコンをクリック
3. プロンプトを選択
4. モード（URL/Text/Capture）を選択
5. 「Geminizeする」ボタンをクリック

### カスタムプロンプトを作成
1. 拡張機能アイコンを右クリック → 「オプション」
2. 「新規追加」をクリック
3. プロンプトの詳細を入力:
   - タイトル
   - 対象AI（Gemini）
   - ページコンテンツの取得設定
   - 自動実行設定
   - デフォルトモード
   - ショートカットキー（任意）
   - プロンプト内容
4. 「保存」をクリック

### ショートカットキーを設定
1. オプション画面でプロンプトを編集
2. 「ショートカットキー」フィールドをクリック
3. 設定したいキーの組み合わせを押す
4. 競合がある場合は警告が表示されます
5. 「保存」をクリック

## 🔧 技術仕様

### 対応ブラウザ
- Google Chrome（Manifest V3対応）
- Microsoft Edge（Chromiumベース）
- Brave
- その他Chromiumベースのブラウザ

### 必要な権限
- `activeTab`: 現在のページのURL・タイトル取得
- `storage`: 設定の保存
- `scripting`: コンテンツ抽出
- `notifications`: 通知表示

### 使用技術
- Manifest V3
- Vanilla JavaScript
- Chrome Extension APIs
- Google Gemini API（ユーザー側）
